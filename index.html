Lenguajes de Programación

Lenguaje de programación se refiere al lenguaje de computadora. Utilizados para desarrollar programas de software, aplicaciones, páginas webs, scripts u otros conjuntos de instrucciones para que sean ejecutadas por los ordenadores.

HTML5

El HTML5 (HyperText Markup Language), es un lenguaje de etiquetas, utilizado para la estructuración y la presentación de contenido en los sitios web. Es utilizado para la estructuración y la presentación de contenido en los sitios web.

CSS3

CSS (Cascading Style Sheets) es lo que se denomina lenguaje de hojas de estilo en cascada y se usa para estilizar elementos escritos en un lenguaje de marcado como HTML. Permite generar el diseño visual de páginas web e interfaces de usuario.

JAVA

Java es uno de los lenguajes más usados y suele ser el empleado como principal por la mayoría de programadores. Numerosas aplicaciones en diferentes campos se han escrito en este lenguaje, relacionados desde la ciberseguridad hasta los videojuegos.

PHP

PHP es un lenguaje de programación que permite el desarrollo web o aplicaciones web dinámicas, el cual es apto para incrustar el lenguaje HTML. Además, el lenguaje PHP favorece a la conexión entre el servidor y a la interfaz del usuario.

PHYTON

Python es un lenguaje de programación de alto nivel que se utiliza para desarrollar aplicaciones de todo tipo.

SQL

SQL (Structured Query Language), es un lenguaje de programación diseñado para actualizar, obtener, y calcular información en bases de datos relacionales.
