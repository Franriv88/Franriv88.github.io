A Generic Page

FSantoro ARQUITECTURA es un proyecto creado para el arquitecto venezolano Fabrizio Santoro Verón. El cual es egresado de la Universidad Simón Bolívar (U.S.B.) de Carcas, Venezuela.
Esta página busca dar a conocer sus obras realizadas, sus proyectos en curso, sus ideas y bocetos. Así como su biografía y trayectoria personal y académica. E irá adáptando y/o agregando información según los requerimientos del arquitecto.
Siendo un proyecto nuevo para ambos, este se irá actualizando conforme surjan nuevas ideas para la página hasta que se suba con un dominio a internet y sea completamente funcional.

Los recursos empelados para la realización son: HTML, CSS3 y Bootstrap, con vistas a incluir próximamente Base de Datos (SQL) y PHP para el lado del servidor (server side). Mediante el uso de FlexBox se implementó el modo responsive haciéndola capaz de adapatars a diferentes dispositivos como Tablets y Celulares, así como a diferentes tamaños de pantallas para computadoras.
